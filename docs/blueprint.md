# **App Name**: Project Forge

## Core Features:

- Landing Page: Introductory landing page to describe the project and available options.
- Project Categories: Dedicated sections for Microcontroller and Minicomputer projects. Microcontrollers: Arduino UNO, PIC, ARM Cortex, ESP32, ESP8266, MSP430, STM32. Minicomputers: DEC PDP-8, DEC PDP-11, Data General Nova, HP 3000 Series, Intel NUC, all types of Raspberry Pi.
- Project Listings: Display a listing of available projects with titles, descriptions, and difficulty levels.
- Project Details: Detailed page for each project with step-by-step instructions, required components, and example code.
- Search and Filter: Enable users to search and filter projects based on keywords, difficulty, components, or other criteria.
- AI Project Suggester: AI tool to suggest projects based on user-selected criteria like skill level, component availability and desired outcome.

## Style Guidelines:

- Primary color: Midnight Blue (#2C3E50) to convey technical expertise and sophistication.
- Background color: Light Gray (#F4F4F8) to provide a clean and distraction-free backdrop.
- Accent color: Electric Blue (#7DF9FF) to highlight interactive elements and calls to action.
- Headline font: 'Space Grotesk' (sans-serif) for a techy, modern feel, best used in page headers and prominent titles.
- Body font: 'Inter' (sans-serif) for body text to ensure readability.
- Use clean, line-style icons to represent different project categories, components, and actions.
- A grid-based layout with clearly defined sections for easy navigation.
- Subtle animations for page transitions and interactive elements to enhance user engagement.
